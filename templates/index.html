Raspberry Pi 3 Control Panel

This is the webpage for controlling the Raspberry Pi 3 to manipulate the output of some led strips with a simple user-friendly interface.

Led strip

The Raspberry Pi is connected to a ws2812b led strip. With 100 individually adressable leds. Those leds are controlled from a python script using the rpi_ws281x library.

Led ring

The Raspberry Pi is also connected to a 8 led RGB ring. Wich are also individually adressable with the same library.

Animations

Using this web you can program either the led strip or the led ring to do certain animations or display a single color/gradient.

Music reaction

This web also allows you to create music reactive animations.
